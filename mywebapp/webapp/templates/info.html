About Group Music

What It Does

Group Music is a web application I created to allow collaboration on Spotify playlists. Playlists on Spotify can be shared and edited by multiple people natively, but this projects takes a different approach on playlist collaboration.
One of the major differences which separates Group Music is the limit on songs per user. This allows a group to evenly create a playlist, without the worry about how many songs each person adds. It also makes it much easier for each user to keep track of what songs they have added and replace their songs.
Group music also allows users who do not use Spotify regularly to interact with the collaborative playlist easily. A Spoitfy account is not required for use, and users use Group Music's simple interface rather than needing to learn to use Spotify's interface.
Group Music is also entirely web based, so nothing needs to be downloaded to use.

Technology

Group Music uses Python and Python-Flask for the server side actions, while the user interface is designed with standard HTML, CSS, and Javascript.

  • Flask
  • Flask-WTF
  • Flask-SQLAlchemy
  • Flask-Migrate
  • Spotipy
